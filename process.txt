The YoYo process

You call `yoyo init <project_name>`.

Then you `cd <project_name>`.

Then you craete a file `build.py`
Then you run `yoyo build build.py`

Now create a `main.py`

Now run `yoyo run main.py`. This will let you use your bindings automatically.



YoYo init creates your project and creates `.yoyo/` folder.
What does the .yoyo/ folder contain?
- pixelscript/

Where does this come from?
It is embedded into the yoyo program.
